Final Term Project: A Simple Chat Application

Project Overview:

We are going to build a simple PHP and MySQL Chat Application using PHP WebSocket Ratchet. This project will include one-to-one chat functionalities with real-time communication powered by the Ratchet WebSocket library.

Features of Group Chat Application:

  • Once a user sends a chat message, other connected users will receive it in real-time using the Ratchet client-server model.
  • User's online or offline status will update after a page refresh.

Features of One-to-One Chat Application:

  • Send one-to-one chat messages in real-time using the Ratchet WebSocket Library.
  • Display or hide unread message notifications in real-time.
  • Display user online or offline status in real-time.
